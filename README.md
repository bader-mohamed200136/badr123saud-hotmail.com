# Sales Performance Dashboard (Excel)

## Project Overview
This project demonstrates the use of Microsoft Excel to analyze sales data and create an interactive dashboard for business decision-making. The dashboard provides insights into sales performance, customer behavior, and product trends through dynamic visualizations and KPI reporting.

## Objectives
- Analyze sales performance and revenue trends
- Identify top-performing products and customers
- Track key business metrics using KPIs
- Create an interactive dashboard for data exploration

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- KPI Cards

## Process
1. Cleaned and prepared raw sales data
2. Removed duplicates and corrected inconsistencies
3. Created Pivot Tables for data aggregation
4. Built Pivot Charts to visualize trends
5. Added slicers for interactive filtering
6. Designed a dashboard to present key insights

## Key Insights
- Identified top-selling products
- Analyzed customer contribution to revenue
- Evaluated sales distribution across products
- Monitored overall business performance through KPIs
